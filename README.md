# TCC 2024 - UFF Rio das Ostras - Frontend

## Resumo

O projeto descrito se trata de uma aplicação Web que dá ao usuário recomendações de cidades para morar de acordo com as preferências do próprio usuário.

A ideia é que o usuário entre com suas preferências, em grau de prioridade, e a aplicação retorna a lista de cidades que mais se encaixa com tais preferências.

## Tecnologias utilizadas

- HTML/CSS/JavaScript

- VueJS

- VuetifyJS - Módulo de estilo componentizado do Vue

## Rodando a aplicação

Para iniciar a aplicação, basta que o computador possua o NodeJS e o NPM instalados.

Após se certificar que ambos estejam devidamente instalados, basta clonar o repositório e, já na pasta raiz, entrar com `npm install` para que sejam instaladas as dependências que se encontram no arquivo `package.json`. Com todas as dependências instaladas, podemos executar a o comando `npm run dev` para compilar em modo desenvolvimento.
